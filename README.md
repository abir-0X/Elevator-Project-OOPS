# Smart Elevator System Simulator

A console-based simulator of a multi-floor building with multiple elevators, focused on high-quality software design, object-oriented programming (OOP), Low-Level Design (LLD), and modern C++17.

## Folder Structure

```text
SmartElevatorSystem/
├── include/              # Header files (.h)
├── src/                  # Source implementation files (.cpp)
├── tests/                # Unit tests (Google Test)
├── docs/                 # Documentation and UML diagrams
├── assets/               # Project assets and design flows
├── CMakeLists.txt        # Build configuration
├── README.md             # Project overview
└── setup.md              # Local setup guide
```

## Features Implemented

- Multi-elevator, multi-floor support.
- External floor buttons (Up/Down) & internal destination buttons.
- State-based elevator movement logic.
- Optimized nearest-elevator dispatch scheduling.
- Google Test automated test coverage.
- Structured comments using the STAR method.
